# 🤖 AI Knowledge Base Assistant (RAG Chatbot)

An AI-powered knowledge assistant that uses **Retrieval-Augmented Generation (RAG)** to answer questions from documents.

This system enables users to upload files and get accurate, context-aware responses using LLMs and vector search.

---

## 🚀 Features

- 📄 Upload and query documents (PDF, TXT, CSV, DOCX)
- 🧠 Retrieval-Augmented Generation (RAG)
- 🔍 Semantic search using embeddings
- ⚡ Context-aware AI responses
- 🔄 Multiple LLM support (OpenAI, Google, HuggingFace)
- 🧩 Advanced retrievers (Cohere reranker, compression)

---

## ⚙️ How It Works

1. Upload Documents  
2. Split into smaller chunks  
3. Convert chunks into embeddings  
4. Store embeddings in vector database (ChromaDB)  
5. Retrieve relevant chunks based on query  
6. LLM generates final answer using retrieved context
